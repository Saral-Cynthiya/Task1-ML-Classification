# Task 1 – ML Classification Project

## Project Objective
To build and evaluate supervised machine learning classification models.

Built-in Scikit-learn dataset (Breast Cancer Dataset) / Spam Email Dataset.

- Logistic Regression
- Random Forest Classifier

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score
- ROC Curve
- Cross Validation
